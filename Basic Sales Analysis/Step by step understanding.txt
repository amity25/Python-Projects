🟢 PROJECT: Basic Sales Analysis (Python Beginner Project)

👉 Goal:

Understand data
Apply loops + conditions + functions
Think like a data analyst



🟢 STEP 1: Create Dataset
sales_data = [
    {"region": "North", "sales": 100},
    {"region": "South", "sales": 200},
    {"region": "East", "sales": 150},
    {"region": "West", "sales": 300},
    {"region": "North", "sales": 250}
]

🔍 Explanation:
List → []
Stores multiple records (like rows in Excel)
Dictionary → {}
Each row is a dictionary with key-value pairs
"region" and "sales" are keys
String → "North", "South"
Integer → 100, 200

👉 Structure:

List = full dataset
Dictionary = one row


🟢 STEP 2: Print All Data
for row in sales_data:
    print(row)

🔍 Explanation:
For Loop → for row in sales_data
Loops through each item in the list
Variable → row
Stores one dictionary at a time
Function → print()
Displays output

👉 Flow:

Loop runs 5 times (because 5 rows)
Each time, row = one dictionary


🟢 STEP 3: Extract Only Sales Values
for row in sales_data:
    print(row["sales"])

🔍 Explanation:
Dictionary Access → row["sales"]
Extracts value using key "sales"
Loop → goes through all rows
String Key → "sales"

👉 Output:
Only numbers will be printed


🟢 STEP 4: Calculate Total Sales
total = 0

for row in sales_data:
    total += row["sales"]

print("Total Sales:", total)

🔍 Explanation:
Variable → total = 0
Stores running total
Loop → iterates through data
Arithmetic Operator → +=
Adds values to total
Dictionary Access → row["sales"]
Function → print()

👉 Logic:

Start with 0
Add each sales value
Final total stored in total


🟢 STEP 5: Apply Condition
for row in sales_data:
    if row["sales"] > 200:
        print(row)

🔍 Explanation:
Loop → goes through each row
Condition (if statement) → if row["sales"] > 200
Checks if sales is greater than 200
Comparison Operator → >
Dictionary Access → row["sales"]

👉 Logic:

Only rows meeting condition are printed


🟢 STEP 6: Group by Region
region_sales = {}

for row in sales_data:
    region = row["region"]
    sales = row["sales"]

    if region in region_sales:
        region_sales[region] += sales
    else:
        region_sales[region] = sales

print(region_sales)

🔍 Explanation:
Dictionary → region_sales = {}
Stores grouped results
Loop → iterates through data
Variables →
region stores region name
sales stores sales value
Condition → if region in region_sales
Checks if key already exists
Dictionary Update →
+= adds sales if region exists
= creates new key if not

👉 Logic:

First time → create region
Next time → add to existing total

👉 This is manual version of groupby


🟢 STEP 7: Function
def total_sales(data):
    total = 0
    for row in data:
        total += row["sales"]
    return total

print(total_sales(sales_data))

🔍 Explanation:
Function Definition → def total_sales(data):
Creates reusable code block
Parameter → data
Input to function
Variable → total
Loop → iterates through data
Return Statement → return total
Sends result back
Function Call → total_sales(sales_data)

👉 Logic:

Pass data → function processes → returns result


Each concept you used maps to real work:

Concept	           Real Use Case

List	           Dataset
Dictionary	   Row (record)
Loop	           Process all data
Condition	   Filter data
Function	   Reusable analysis
